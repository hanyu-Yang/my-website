# Yang Hengyu's Personal Website

这是我的个人主页，基于 Quarto 构建并托管在 GitHub Pages 上。

## 🌐 访问地址
[https://hanyu-Yang.github.io/my-website/](https://hanyu-Yang.github.io/my-website/)

## 🛠️ 技术栈
- **框架**: Quarto (基于 Markdown)
- **部署**: GitHub Pages (gh-pages 分支)
- **样式**: 自定义 CSS + Quarto 主题

## 📄 网站结构
- `index.qmd`: 首页（个人简介、联系方式）
- `about.qmd`: 关于我（教育背景、技能经历）
- `_quarto.yml`: 网站全局配置
- `styles.css`: 自定义样式
